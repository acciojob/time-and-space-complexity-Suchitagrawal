Time,Space,Complexity

The valid algorithm takes a finite amount of time for execution. The time required by the algorithm to solve given problem is called Time complexity of the algorithm. Time complexity is very useful measure in algorithm analysis and it is very efficient also.

Problem-solving using computer requires memory to hold temporary data or final result while the program is in execution. The amount of memory required by the algorithm to solve given problem is called Space complexity of the algorithm. usually time taken by a program is O(n).
